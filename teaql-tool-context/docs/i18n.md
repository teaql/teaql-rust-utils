# TeaQL Tool API: Internationalization (I18n)

Use `ctx.i18n()` to easily access localized strings and perform translations within the TeaQL runtime context.

> [!NOTE]
> The facade `ctx.i18n()` exposes read-only methods for translation. For mutable operations such as loading translations or setting the default locale, you should construct an `I18nTool` directly.

## Required Dependencies

To use the i18n utilities, you must add `teaql-tool` to your project with the `std` feature enabled.

```toml
teaql-tool = { version = "1.0", features = ["std"] }
```

## Translation Examples

```rust
// Basic translation for a specific locale
let greeting = ctx.i18n().t("en-US", "hello_message");

// Translation with formatting arguments
let welcome = ctx.i18n().tf(
    "en-US",
    "welcome_user",
    &[("name", "Alice"), ("role", "Admin")]
);
```

## Key Methods
- `.t(locale, key)`: Retrieves a simple translation string for the given locale and key.
- `.tf(locale, key, args)`: Retrieves a translation string formatted with the provided arguments (key-value pairs).
